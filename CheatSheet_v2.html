World Wide Web

Die wichtigsten Technologien die das Web zum Erfog gemacht haben

  • Uniform Resource Locator (URL) und Uniform Resource Identifier (URI)
  • HyperText Markup Language (HTML)
  • HyperText Transfer Protocol (HTTP)

URL

Die allgemeinen Bestandteile einer URL sind:

  • Protokoll (z.B. http)
  • Hostname (z.B. www.example.com)
  • Dateiname (z.B. index.html)

Web Server

Ein Webserver ist ein Computersystem, das Anforderungen über HTTP verarbeitet (das grundlegende Netzwerkprotokoll, das verwendet wird, um Informationen über das World Wide Web zu verteilen). Die primäre Funktion eines Webservers ist die Speicherung, Verarbeitung und Bereitstellung von Webseiten an Clients. Die Kommunikation zwischen Client und Server erfolgt über das Hypertext Transfer Protocol (HTTP).

Inhalte werden auf ihren Datenbanken gespeichert. Auf dem Web-Server ist die HTML-Datei, die der Client bekommen kann, damit er die Seite sehen kann. Der Inhalt von Webseiten wird in HTML geschrieben.

Web Client

Webbrowser oder allgemein auch Browser sind spezielle Computerprogramme zur Darstellung von Webseiten im World Wide Web oder allgemein von Dokumenten und Daten. Neben HTML-Seiten können Webbrowser verschiedene andere Arten von Dokumenten wie zum Beispiel Bilder und PDF-Dokumente anzeigen. Webbrowser stellen die Benutzeroberfläche für Webanwendungen dar.

HTML

HTML Elemente

Ein HTML-Element besteht normalerweise aus einem Start-Tag und einem End-Tag, mit dem Inhalt dazwischen.
<tagname>Inhalt... </tagname>

Void Elements

HTML-Elemente ohne Inhalt werden als void elements bezeichnet. Void elements können im Start-Tag geschlossen werden:
z.B.: <br /> (Zeilenumbruch)

DOCTYPE

Die !DOCTYPE Dekleration ist kein HTML-Tag. Sie ist eine Anweisung für den Webbrowser, über welche HTML Version die Seite geschrieben wird. (In HTML 5 sieht das so aus: <!DOCTYPE html>)

Attribute

  • Alle HTML-Elemente können Attribute haben
  • Attribute liefern zusätzliche Informationen zu einem Element
  • Attribute werden immer im Start-Tag angegeben
  • Attribute kommen normalerweise in Name/Wert Paare vor. Wie: name = "value"

Das href-Attribut

HTML-Links werden mit dem <a>-Tag definiert. Die Linkadresse wird im href-Attribut angegeben.
z.B.: <a href="https://www.w3schools.com">Das ist ein Link</a>

Das scr-Attribut

HTML-Bilder werden mit dem <img>-Tag definiert. Der Dateiname der Bildquelle wird im src-Attribut angegeben.

Das alt-Attribut

Das alt-Attribut gibt einen alternativen Text an, der verwendet werden soll, wenn ein Bild nicht angezeigt werden kann.

Das lang-Attribut

Die Sprache des Dokuments kann im <html>-Tag mit dem lang-Attribut deklariert werden.

Das title-Attribut

Das Titelattribut liefert zusätzliche "Tool-Tipp" -Informationen, die angezeigt werden wenn man mit der Maus über den Absatz(oder der Überschrift, ...) fährt.

Der "html tag"

  • Er teilt dem Browser mit, dass es sich um ein HTML-Dokument handelt.
  • Er repräsentiert den Stamm eines HTML-Dokuments.
  • Er ist der Behälter für alle anderen HTML-Elemente (mit Ausnahme des !DOCTYPE tags).

Hervorhebungen eines Textes

  • Der em tag: macht einen Text hervorgehobener. Wird meist kursiv dargestellt
  • Der strong tag: markiert wichtigen Text. Wird fett gedruckt dargestellt
  • Der b tag: macht den Text fett gedruckt

Die beiden wichtigsten HTML Elemente

Das Head Element

Das Head-Element ist ein Container für Metadaten über das HTML-Dokument. Sie werden nicht angezeigt. Metadaten definieren typischerweise den Dokumententitel, den Zeichensatz, die Stile, die Links, die Skripte und andere Metainformationen.

Das Title Element

Es definiert den Titel des Dokuments und ist in allen HTML/XHTML-Dokumenten erforderlich.

Das Link Element

Es wird verwendet, um mit externen Stylesheets zu verknüpfen.
z.B.: <link rel="stylesheet" type="text/css" href="StyleSheet.css">

Das Meta Element

Mit ihm werden Zeichensätze, Seitenbeschreibung, Schlüsselwörter, Autor und andere Metadaten festgelegt. Beispiele:
<meta charset="UTF-8"> <meta name="description" content="Free Web tutorials"> <meta name="keywords" content="HTML,CSS,XML,JavaScript"> <meta name="author" content="John Doe">

Das Body Element

Es enthält alle Inhalte eines HTML-Dokuments wie Text, Hyperlinks, Bilder, Tabellen, Listen usw.

Weitere wichtige HTML Elemente

Der Artikel

Das Artikel ELement gibt unabhängige, in sich geschlossene Inhalte an. Ein Artikel sollte selbstverständlich sein und es sollte möglich sein, ihn unabhängig vom Rest der Seite zu verteilen. Syntax: 

					<article>
						content
					</article>

Zeilenumbruch

Verwenden Sie <br/>, wenn Sie einen Zeilenumbruch (eine neue Zeile) wollen, ohne einen neuen Absatz zu starten.

Überschriften

Überschriften sind mit den <h1> bis <h6> tags definiert. <h1> wichtigste Überschrift. <h6> am wenigsten wichtigste Überschrift. z.B.: <h1>Überschrift 1</h1>

<hr/> definiert eine thematische Veränderung des Inhalts. Wird als horizontale Linie dargestellt.

Paragraphen

Das HTML <p>-Element definiert einen Absatz. Syntax: 

					<p>
						Inhalt
					</p>

<pre> definiert vorformatierten Text.

Bilder

In HTML werden Bilder mit dem <img>-Tag definiert. Es enthält nur Attribute und hat keinen Schluss-Tag. Das src-Attribut gibt die URL (Webadresse) des Bildes an. alt-Attribut: siehe Attribute.
Syntax: 

					<img scr="url" alt="irgendein Text"/>

Section

Das <section>-Tag definiert Abschnitte in einem Dokument, wie z. B. Kapitel, Kopfzeilen, Fußzeilen oder andere Abschnitte des Dokuments.
Syntax: 

					<section>
						Inhalt
					</section>

Links

HTML-Links sind Hyperlinks. Man kann auf einen Link klicken und zu einem anderen Dokument springen.
Syntax: 

					<a href="url">link text</a>

Bookmarks: Zuerst muss man ein Bookmark mit dem id-Attribut machen. z.B.: <h1 id="c1">Chapter 1</h1> Dann beim href den Bookmark eingeben und du hast eine Verlinkung zu der Überschrift.

Kommentare

Kommentar-Tags werden verwendet, um Kommentare in den HTML-Quellcode einzufügen. Syntax: 

					<!-- Write your comments here -->

CSS

CSS Syntax

Ein CSS-Regelsatz besteht aus einem Selektor und einem Deklarationsblock. Der Selektor zeigt auf das HTML-Element, das man stylen möchte. Der Deklarationsblock enthält eine oder mehrere Deklarationen, die durch Semikolons getrennt sind. Jede Deklaration enthält einen CSS-Eigenschaftsnamen und einen Wert, der durch einen Doppelpunkt getrennt ist. Eine CSS-Deklaration endet immer mit einem Semikolon, und Deklarationsblöcke sind von geschweiften Klammern umgeben. 

					Selector {															
						Property: Value;					
					}									 
				
				   Bsp:	h1 {
							color: blue;
						}

Gruppierte Selektoren

Wenn man Elemente mit den gleichen Stildefinitionen hat, ist es besser, die Selektoren zu gruppieren, um den Code zu minimieren. Um Gruppenselektoren zu gruppieren, trennt man jeden Selektor mit einem Komma. z.B.: 

					h1,h2,h3 {
						color: red;
					}

CSS Kommentare

					/* This is a single-line comment */

Farben

Farbnamen: In HTML kann eine Farbe mit einem Farbnamen angegeben werden. Diese Farben gibt es

In HTML kann eine Farbe mit einem Hexadezimalwert in dieser Form angegeben werden:
#rrggbb
Wo rr (rot), gg (grün) und bb (blau) hexadezimale Werte zwischen 00 und ff (wie dezimal 0-255) sind. Beispielsweise wird #ff0000 als rot angezeigt, da Rot auf den höchsten Wert (ff) gesetzt ist und die anderen auf den niedrigsten Wert (00).

Hintergründe

background-color Property

Der background-color Property gibt die Hintergrundfarbe eines Elements an. Syntax: 

						background-color: colorname;

background-image Property

Syntax: 

						background-image: url("image-url");

Wiederhole Horizontal oder Vertikal

syntax: für horizontales Wiederholen 

							background-repeat: repeat-x;

Für vertikales Wiederholen x mit y ersetzen (repeat-y).

Fixieren

Um festzulegen, dass das Hintergrundbild fixiert werden soll (nicht mit dem Rest der Seite mitscrollen), verwendet man die Eigenschaft background-attachment. 

							background-attachment: fixed;

Text

Text Farbe

Der color Property wird verwendet, um die Farbe des Textes festzulegen. z.B.: 

						h1 {
							color: green;
						}

Textausrichtung

Der text-align Property wird verwendet, um die horizontale Ausrichtung eines Textes festzulegen. Ein Text kann links oder rechts ausgerichtet, zentriert oder justified sein.

  • Links: text-align: left;
  • Rechts: text-align: right;
  • Zentriert: text-align: center;
  • justified: text-align: justify;
Wenn die Textausrichtungseigenschaft auf justified gesetzt ist, wird jede Zeile so ausgedehnt, dass jede Zeile gleich breit ist und die linken und rechten Ränder gerade sind (wie in Zeitschriften und Zeitungen).

Text Dekoration

Der text-decoration property wird verwendet, um Dekorationen im Text zu setzen oder zu entfernen.

  • text-decoration: none;
  • text-decoration: overline;
  • text-decoration: line-through;
  • text-decoration: underline;

Textumwandlung

Der text-transform property wird verwendet, um Groß- und Kleinbuchstaben in einem Text anzugeben. Es kann verwendet werden, um alles in Groß- oder Kleinbuchstaben umzuwandeln oder den ersten Buchstaben jedes Wortes großzumachen.

  • text-transform: uppercase;
  • text-transform: lowercase;
  • text-transform: capitalize;

Fonts

Die CSS font properties definieren die Schriftfamilie, die Kühnheit, die Größe und den Stil eines Textes.

Unterschied zwischen Serif und Sans-Serif Fonts

Serif-Schriften haben kleine Linien an den Enden bei einigen Buchstaben.
Sans-serif Fonts haben keine Linien an den Enden der Buchstaben.

Font Family

Die Schriftfamilie eines Textes wird mit dem font-family Property gesetzt. Sie sollten mehrere Fontnamen als "Fallback" -System enthalten. Wenn der Browser die erste Schrift nicht unterstützt, versucht er die nächste Schriftart und so weiter. Beginnen Sie mit der Schriftart, die Sie wollen, und enden mit einer generischen Familie, um den Browser eine ähnliche Schriftart in der generischen Familie zu wählen, wenn keine anderen Schriftarten verfügbar sind.

Anmerkung: Wenn der Name einer Schriftfamilie mehr als ein Wort ist, muss es in Anführungszeichen sein, wie zB: "Times New Roman". Mehr als eine Schriftfamilie wird in einer durch Kommas getrennten Liste angegeben. z.B.: 

						font-family: "Times New Roman", Times, serif;

Font Style

Die Eigenschaft font-style wird hauptsächlich verwendet, um kursiven Text anzugeben. Diese Eigenschaft hat drei Werte:

  • normal - Der Text wird normal angezeigt
  • italic - Der Text wird kursiv dargestellt
  • oblique - Der Text ist "lehnend" (oblique ist sehr ähnlich italic, aber wird weniger unterstützt)
z.B.: 

						font-style: italic;

Font Size

Der font-size Property legt die Größe des Textes fest. Der font-size Wert kann eine absolute oder relative Größe sein.
Absolut Size:

  • Setzt den Text auf eine angegebene Größe
  • Erlaubt einem Benutzer nicht, die Textgröße in allen Browsern zu ändern (schlecht für Zugänglichkeitsgründe)
  • Absolute Größe ist nützlich, wenn die physikalische Größe der Ausgabe bekannt ist
Relative size:
  • Legt die Größe relativ zu den umgebenden Elementen fest
  • Ermöglicht einem Benutzer, die Textgröße in Browsern zu ändern
Die Einstellung der Textgröße mit Pixeln gibt Ihnen die volle Kontrolle über die Textgröße. Damit Benutzer die Größe des Textes ändern können (im Browser-Menü), verwenden viele Entwickler em statt Pixel. (1em = 16px: pixels/16=em) Syntax.: 

						font-size: 2.5em; /* 40px/16=2.5em */

Font Weight

Der font-weight Property gibt das Gewicht einer Schriftart an. (normal oder bold!). 

						font-weight: bold;

Link gestallten

Links können mit jedem CSS Property gestalltet werden(z.B. color, font-family, background, etc.). Darüber hinaus können Links unterschiedlich gestylt werden, je nachdem, in welchem ​​Zustand sie sich befinden. Die vier Linkszustände sind:

  • a:link - ein normaler, nicht besuchter Link
  • a:visited - ein Link, den der Benutzer besucht hat
  • a:hover - ein Link, wenn der Benutzer mit der Maus darüber fährt
  • a:active - ein Link, wenn er gerade geklickt wird
Bei der Einstellung des Stils für mehrere Link-Zustände gibt es einige Regeln:
  • a:hover muss nacha:link und a:visited kommmen
  • a:active muss nach a:hover kommen
z.B.: 

					/* unvisited link */
					a:link {
						color: red;
					}

					/* visited link */
					a:visited {
						color: green;
					}

					/* mouse over link */
					a:hover {
						color: hotpink;
					}

					/* selected link */
					a:active {
						color: blue;
					}

CSS Kombinatoren

Ein CSS-Selektor kann mehr als einen einfachen Selektor enthalten. Zwischen den einfachen Selektoren können wir einen Kombinator einschließen. Ein Kombinator ist etwas, das die Beziehung zwischen den Selektoren erklärt. Es gibt vier verschiedene Kombinatoren in CSS3:

  • descendant selector (Leertaste)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Descendant selector

Der Nachkommenselektor passt zu allen Elementen, die Nachkommen eines bestimmten Elements sind. Im folgenden Beispiel werden alle <p> Elemente innerhalb von <div> Elementen ausgewählt: 

						div p {
							background-color: yellow;
						}

Child Selector

Der Kinderselektor wählt alle Elemente aus, die die unmittelbaren Kinder eines bestimmten Elements sind. Im folgenden Beispiel werden alle <p>-Elemente ausgewählt, die unmittelbare Kinder eines <div>-Elements sind: 

						div > p {
							background-color: yellow;
						}

Adjacent Sibling Selector

Der benachbarte Geschwisterselektor wählt alle Elemente aus, die die benachbarten Geschwister eines bestimmten Elements sind. Geschwisterelemente müssen das gleiche übergeordnete Element haben, und "benachbart" bedeutet "unmittelbar nach". Im folgenden Beispiel werden alle <p> Elemente ausgewählt, die unmittelbar nach <div> Elementen platziert werden: 

						div + p {
							background-color: yellow;
						}

General Sibling Selector

Der allgemeine Geschwisterselektor wählt alle Elemente aus, die Geschwister eines bestimmten Elements sind. Das folgende Beispiel wählt alle <p> Elemente, die Geschwister von <div> Elementen sind: 

						div ~ p {
							background-color: yellow;
						}

Pseudo Klassen

					selector:pseudo-class {
						property:value;
					}

z.B.: Bei Links: a:active.
